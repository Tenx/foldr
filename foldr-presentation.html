Where 1984 Meets 2026

Foldr

Manage AI agents like you manage folders

A nostalgic recreation of Macintosh System 1 (1984)
for the agentic future

The Problem

Modern AI agent tools feel like
rocket science

  • Complex dashboards with too many options
  • Steep learning curves for non-technical users
  • Abstract interfaces that don't map to mental models
  • Lost in nested menus and configuration screens

What if agent management
felt familiar?

Drag. Drop. Done.

Foldr brings back the desktop metaphor:
Folders are projects. Icons are agents.
Double-click to open. Drag to organize.

Authentic Retro,
Modern Power

🖥️
Mac System 1 Aesthetic
1-bit monochrome, pixelated fonts, hard edges
📁
Desktop Metaphor
Folders, icons, draggable windows
🤖
Agentic OS Concept
Departments as agents, parent-child relationships

Three-Layer Hierarchy

  • Desktop: All your companies/projects as folder icons
  • Company: Department agents on a draggable canvas
  • Department: Sub-agents with skills, tools, and memory

Drag one agent onto another to create relationships.
Just like organizing files in 1984.
